 Dépendances installées :
✅ Express → Crée le serveur
✅ Mongoose → Connecte MongoDB
✅ Bcrypt → Hachage des mots de passe
✅ JWT → Authentification
✅ Cors → Communication backend/frontend
✅ Dotenv → Variables d’environnement

⚙️ 1️⃣ Configuration du Backend
Va dans le dossier backend et initialise un projet Node.js :
-----------------
cd backend
npm init -y
npm install express mongoose bcrypt jsonwebtoken cors dotenv
------------------------

📌 Explication :
- express → Crée le serveur.
- mongoose → Gère MongoDB.
- bcrypt → Hache les mots de passe.
- jsonwebtoken → Authentification avec JWT.
- cors → Autorise les requêtes entre le frontend et le backend.
- dotenv → Gestion des variables d’environnement.
